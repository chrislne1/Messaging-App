# Messaging App (Facebook-Inspired)

This project is a full-stack messaging application inspired by core features of Facebook Messenger.  
It is being developed as a learning-focused project to understand how modern web applications handle user management, APIs, and real-time-style communication using a clean client‚Äìserver architecture.

The app is currently in early development and focuses on building a solid backend foundation before expanding to advanced features such as authentication and real-time messaging.

---

## Current Features
- User creation and storage
- REST API for user data
- Clean separation between client and server
- Centralized API service layer on the frontend
- Database-backed persistence using PostgreSQL

---

## Tech Stack

### Frontend (Client)
- **React**
- **JavaScript**
- **HTML / CSS**

### Backend (Server)
- **Node.js**
- **Express.js**
